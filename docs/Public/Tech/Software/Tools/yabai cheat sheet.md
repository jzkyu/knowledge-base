Change window focus within space:
- `alt` + `j/k/h/l`

Swap windows:
- `shift` + `alt` + `j/k/h/l`

Change focus between external displays:
- `alt` + `s/g`

Move window to different display:
- `shift` + `alt` + `s/g`

Maximize a window:
- `shift` + `alt` + `m`

Balance windows:
- `shift` + `alt` + `e`

Start / stop / restart yabai:
- `ctrl` + `alt` + `q/s/r`

Rotate layout clockwise:
- `shift` + `alt` + `r`
